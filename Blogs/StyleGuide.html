Website Style Guide Reflection

According to Launch Marketing, a website style guide is "a document that provides guidelines for the way your brand should be presented from both a graphic and language perspective." (Launch Marketing 2018). From my understanding, a style guide gives users and targeted audiences an illustration of how to identify your brand, moreover, how to interpret it.

My website style guide is divided into 3 main categories that make up my website. The first category is the colour palette. In my colour palette, I have demonstrated all the colours I have used throughout my website including in the circular packing diagram. The colours are used in groups for consistency, the navigation bar makes use of the darker colours on the right, the diagrams and data visualisations make use of the bright and lively colours on the left, and the remaining colours are used throughout the website.

The next category reflected in my website style guide is font usage. I have a total of 5 fonts throughout the website. Cody star is the chosen font for the landing page. I chose this font for the landing page as it provides a techy feel to the data visualisation illustration that is on the landing page. Khand is the next font that I have used throughout the website as a default font, it is used on the about page, blog posts, and the reference page. I chose this font as it provides a sophisticated feel to the layout of the website.

Kalam is a font that I have used only for the data visualisation to provide users with the cue of them accessing a different atmosphere of the website, being the creative section. Arial is the font that I used to communicate the diagram keys. This is to ensure it can be easily read by all individuals to avoid misinterpretation as a result of not being able to read the information. The fifth font has not been used as yet, this is the Srisakdi font, which is very playfully, I aim to incorporate this font into future data visualisation.

The final section of my website style guide provides a visual demonstration to the mood and atmosphere I hope to achieve, the mood board. The mood board is meant to reflect the type of visual presentation that is to be used. My mood board reflects purples and lights, for brightness, which is how I aim to present my graphs.
